# Texture-Pack-Of-Doom-And-Despair
terrible and awful texture pack created from requests on tiktok.

=======DOWNLOAD INSTRUCTIONS=======

(You will need the Texture Loader mod off geode)

STEP ONE: If you havennt already, download the Texture Loader mod on Geode.

STEP TWO: On this page, go to Releases, then go to the texture pack. It should be called "Texture Pack Of Doom And Despair" There should be three blue links under the description. Download the link at the very top by clicking on it.

STEP THREE: Once its downloaded, you should have the .zip file in your downloads. Now open Geometry Dash.

STEP FOUR: Go to your settings tab. Press graphics. At the top right corner there should be two tabs saying "Advanced" and "Textures"

STEP FIVE: Click on "Textures"

STEP SIX: A popup should come up. There is an image of a file in the bottom right corner. Click on it.

STEP SEVEN: This will bring up your texture files. Open file explorer a second time and drag the .zip file into the texture file area.

Thats it! Once you open Geometry Dash and refresh the texture page, the textures should load up. Drag the textures to "Applied"

One last thing: DO NOT EXTRACT FILES. Texture Loader only knows how to read .zips so that folder will be useless

Enjoy!
